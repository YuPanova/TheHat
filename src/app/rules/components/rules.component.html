Правила игры

  1. Команды: Разделите участников на команды. Рекомендуется, чтобы каждая команда состояла из нескольких человек. Введите имена игроков через запятую в соответствующее поле.
  2. Заполнение слов: В данной игре сами игроки вводят слова, которые будут отгадывать. По очереди каждый игрок ввоит определенное в настройках количество слов. Каждое слово проходит проверку на грамматику (если есть подключение к интернету), а также проверяется на отсутствие дубликатов.
  3. Выбор описывающего: Каждый раунд один из членов команды становится описывающим. Этот игрок будет описывать слова своей команде без использования однокоренных слов, не используя показывая на предметы вокруг.
  4. Отгадывание: Команда должна попытаться угадать, о каких словах идет речь, и называть их вслух. Описывающий может переходить к следующему слову, как только команда угадывает текущее или пропустить данное слово, но тогда оно не будет засчитано.
  5. Засекание времени: Играйте в несколько раундов. У каждой команды есть ограниченное время на угадывание слов. На экране будет виден отсчет таймера, после окончания времени издастся звук.
  6. Подсчет баллов: За каждое успешно угаданное слово команда получает один балл.
  7. Смена ролей: После каждого раунда смените роли внутри команды. Каждый участник по очереди будет описывающим. На экране высветиться имя следующего описывающего слова.
  8. Победитель: Та команда, которая наберет больше баллов, становится победителем. Конец игры по истечению все заганных слов.
